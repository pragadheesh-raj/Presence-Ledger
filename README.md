# Presence Ledger

Presence Ledger is a simple, structured system for recording and managing daily workforce presence. It is designed to provide a clear and consistent view of attendance without unnecessary complexity.

---

## Overview

The system enables teams to:

- Record daily attendance
- Maintain a centralised presence log
- View and manage records through a simple interface
- Ensure consistency in attendance tracking across the organisation

Presence Ledger focuses on reliability and clarity rather than feature overload.

---

## Features

- Daily attendance recording
- Structured presence logs
- Lightweight interface
- Easy deployment and setup
- Suitable for small teams and internal use

---

## Tech Stack

- Node.js

- Frontend (HTML / CSS / JS or your framework)
